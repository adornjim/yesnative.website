# YES NATIVE - E-commerce Wellness Store

A premium millet-based superfoods e-commerce platform built using React, Vite, Tailwind CSS, and Shadcn UI. 

This application provides clean-label, functional nutrition products designed for modern lifestyles, specializing in weight management, family health, women's wellness, and kids nutrition.

## Features

- **Dynamic Homepage:** Beautiful interactive sections highlighting wellness goals, best sellers, brand story, and customer reviews.
- **Shop Catalog:** An fully featured shop catalog showing premium superfood blends with interactive filtering and cart addition.
- **Millet Wellness Goals:** Goals selector to match individual health requirements with targeted millet superfoods.
- **Cart Management:** Responsive, fully interactive in-memory shopping cart with item quantity adjustments and checkout overview.
- **High Performance:** Standard CSS scroll-triggered animation system and hardware-accelerated micro-interactions.
- **Premium Aesthetics:** Curated typography, harmonious custom HSL palettes, and immersive glassmorphic cards.

## Setup & Running

To run this project locally, follow these steps:

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start Local Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

## Technology Stack

- **Framework:** React 18 & Vite
- **Styling:** Tailwind CSS & Shadcn UI (Radix primitives)
- **Icons:** Lucide React
- **Animations:** Custom scroll observer & GPU-optimized keyframe transitions
